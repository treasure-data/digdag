Release 0.8.4
=============

General changes
---------------

* The td>, td_load> and td_for_each> operators are now async polling instead of blocking.
* Model validation bug fixes.


Server mode changes
-------------------

* Model validation errors now return 400 Bad Request instead of 500 Internal Error.
